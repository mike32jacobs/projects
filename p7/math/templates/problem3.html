Sometimes you think that you have solved a problem, but you have not. I used mathml to display a system of equations. It seemed to work the solution worked in my Canvas LMS instance, but it was not working on this web application that you are viewing. With a little help from Thomas Ballatore, I realized that it worked on Safari, and it worked within Canvas on Chrome. It did not work in this web application on Chrom. It turns out that Chrome does not support mathml. Canvas does some other web magic to make it appear correctly.

My goal is to do some of that web magic here.

There are a few possible solutions to this problem, but each has it's costs.

  • Ask users to use a specific browswer. The problem with this solution is that the teacher may not have the ability to influence a student's browser choice. In my school, students use Chromebooks, so using Safari is not an option.
  • Use images. You can take a screenshot of the system of equations. This will work, but if you do it frequently, it could be a time consuming process. A look at Problem 2 will show you how to take a screenshot, and then upload a desription into the Canvas LMS. Making the image accessible is important, but also time-consuming
  • Use Mathjax to do this for you. This is a very effective solution, but it requires a basic knowledge of html, and javascript. Althouth the solution is effective, it is not reasonable to expect that all math teachers will be willing and able to add this to their tool box.
